# Python Calculators & Utilities

**Language:** Python  
**Type:** General Learning / Utilities

## Description
Collection of Python calculator tools and utility scripts developed during cybersecurity course.

## Features
- Calculator implementations
- Subnetting calculator
- Network utilities
- Problem-solving exercises

## Files
- `homework_lesson19.py` - Lesson exercises
- `maximum_calculator_with_additional_option.py` - Advanced calculator
- `subnetting_calculator_mainFunction.py` - Network subnetting tool

## Usage
```bash
python homework_lesson19.py
python subnetting_calculator_mainFunction.py
```

## Purpose
Learning tools for understanding Python programming and networking concepts.
